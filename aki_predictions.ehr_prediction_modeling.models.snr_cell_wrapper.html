Definition of SNR wrapper for RNN cells.

class aki_predictions.ehr_prediction_modeling.models.snr_cell_wrapper.SNRWrapper(cell, cell_type: RNNCellType, tasks: List[str], dropout_is_training: bool, snr_config: ConfigDict, parallel_iterations: int, should_route: bool = True, use_task_specific_routing: bool = True, connection_name: str = 'routing')ÔÉÅ

Bases: AbstractModule

A wrapper around a ResetCore, part of a subnetwork routing architecture.

This wrapper takes care of creating SNRConnections from each input in a list of inputs and feeding the combined result into the cell it wraps. In this architecture the cell will typically be a ResetCore.

If the inputs passed are not a list, but a single tensor, then it will keep those as they are and pass them on.
