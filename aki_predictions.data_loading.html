aki_predictions.data_loading.datetime_bins(date_time, bins: int, include_minutes=False, string_format=None) → int

Takes a datetime string and returns bin index.

An additional bin (n_bins + 1) is added to account for all events which do not have a specific time element. Times matching 00:00:00 are assumed to be those without specific time element.

Parameters:

  • date_time (str) – Date and time in the format ‘%d/%m/%Y %H:%M:%S’.

  • bins (int) – Number of bins.

  • include_minutes (bool) – Optional, if True then calculates bin position by minutes.

Returns:

Bin index between 0 and bins + 1

Return type:

(int)
