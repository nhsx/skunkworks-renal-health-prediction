mcc() → floatÔÉÅ

Compute the Matthews Correlation Coefficient for the predicitons.

MCC is a useful metric for imbalanced datasets. It is a correlation coefficient between the observed and predicted (binary) classifications.

NOTE: If we move to the multiclass case, this is also possible to generalise as an Rk statistic.

Returns:

The value of the MCC for the provided predictions.
